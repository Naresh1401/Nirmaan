# Business Model — Nirmaan

## Revenue Streams

### 1. Marketplace Commission (Primary Revenue)

| Category | Commission Rate | Example |
|----------|----------------|---------|
| Cement, Steel | 2–3% | ₹500 on ₹20,000 order |
| Sand, Gravel, Bricks | 3–5% | ₹750 on ₹15,000 order |
| Tiles, Fittings | 5–8% | ₹1,200 on ₹20,000 order |
| Paint, Electrical | 5–10% | ₹1,500 on ₹15,000 order |

**Year 1 Target**: Average 3.5% commission on ₹15,000 AOV = **₹525/order**

### 2. Logistics Coordination Fee

| Order Value | Delivery Fee |
|-------------|-------------|
| < ₹5,000 | ₹300 |
| ₹5,000 – ₹25,000 | ₹500–₹800 |
| ₹25,000 – ₹1,00,000 | ₹800–₹1,200 |
| > ₹1,00,000 | ₹1,200–₹1,500 |

Platform retains 20–30% of delivery fee; rest goes to delivery partner.

### 3. Supplier Subscription (SaaS Revenue)

| Plan | Monthly Fee | Features |
|------|-------------|----------|
| Basic | Free | Listing, order management |
| Silver | ₹2,000/mo | Featured listing, analytics dashboard |
| Gold | ₹5,000/mo | Priority placement, bulk order access, dedicated support |
| Enterprise | Custom | API integration, multi-branch management |

### 4. Bulk Procurement Services

For large contractors/developers managing multi-crore projects:
- **Procurement management fee**: 1–2% of total project material value
- **Dedicated account manager**
- **Custom pricing negotiations with suppliers**
- **Scheduled delivery management**

### 5. Financial Services (Phase 3+)

| Service | Revenue Model |
|---------|--------------|
| Supplier Credit | Interest spread (2–4% over lending rate) |
| Contractor BNPL | Processing fee + interest |
| Insurance | Commission from insurance partners |
| Payment Processing | 0.5% payment gateway margin |

## Unit Economics

### Per Order Economics

| Metric | Value |
|--------|-------|
| Average Order Value (AOV) | ₹15,000 |
| Marketplace Commission (3.5%) | ₹525 |
| Logistics Fee (platform share) | ₹200 |
| **Gross Revenue per Order** | **₹725** |
| Customer Acquisition Cost | ₹150 (blended) |
| Operations Cost per Order | ₹100 |
| Tech Cost per Order | ₹25 |
| **Net Contribution per Order** | **₹450** |
| **Contribution Margin** | **62%** |

### Monthly Projections (Year 1 – Peddapalli)

| Month | Orders/Day | Monthly Revenue | Monthly Cost | Net |
|-------|-----------|----------------|-------------|-----|
| M1–M3 | 10 | ₹2.17L | ₹5L | -₹2.83L |
| M4–M6 | 30 | ₹6.5L | ₹6L | ₹0.5L |
| M7–M9 | 60 | ₹13L | ₹8L | ₹5L |
| M10–M12 | 100 | ₹21.7L | ₹10L | ₹11.7L |

### Break-even Analysis

- **Break-even point**: ~40 orders/day
- **Expected timeline**: Month 5–6
- **Burn rate (pre-revenue)**: ₹5–7L/month
